# Bloc Chat

## What is it?

Bloc Chat is a real-time chat application built with AngularJS and uses the Firebase Backend-as-a-Service.

**Languages/Frameworks/Libraries/Tools:**
* HTML
* CSS
* Javascript
* AngularJS
* jQuery
* NodeJS
* Bootstrap
